Description:
I completed this project while learning the basics of web development. This project was the final culmination of Colt Steel's "The Web Developer Bootcamp". For this project I used Express as my framework and ejs as my framework. For the backend I utilized Nodejs and applied RESTful API for the routes while troubleshooting using Postman. Description:
In grade 11 I took an intro to computer science class and for the culminating I decided to make a game. Survive! is a 2D top-down survival shooter game. I used fundamental object oriented programming techniques to implement the game physics and features with the Swing library. One of these features for example would be the camera and map that would follow the player around while moving. Additionally, I also used a tilesheet for obtaining the textures and applying them to the mobs and backgrounds.

Description:
A revisualization of the time-old board game Yahtzee! This game was made with the assistance of Colt Steel's "The Modern React Bootcamp". This was a learning process for me as I had never worked with a framework like react before. Initially I found that the javascript utilized in it was daunting. However, I eventually got used to it and was able to create this game whilst maintaining the modularity of the code. I really enjoyed making the score keeping cards for this project as it involved merging vanilla JS with React.

Description:
LightsOut is a board game played by one player, similar to my yahtzee game. This is one of the earlier projects I created with React; also with the help of Colt Steel's "The Modern React Bootcamp". In this game I learned how to control props and states efficiently. Additionally it was one of my first experiences applying css React. I found it jarring that it was the standard to create a new css file for each component, but eventually I came around to it and now appreciate how organized it makes everything. I really liked the styling done on this game, especially the animation on the title.
